Class NonVolatileColorVisualizer

  • java.lang.Object
    • ca.eandb.jmist.framework.display.visualizer.NonVolatileColorVisualizer
  • All Implemented Interfaces:

    Direct Known Subclasses:
    AutomaticLinearChannelVisualizer

    public abstract class NonVolatileColorVisualizer
extends java.lang.Object
implements ColorVisualizer
    A ColorVisualizer that never changes its visualization function asynchronously (that is, one that does not fire state-change events). A ColorVisualizer of this type may only change its visualization function when ColorVisualizer.analyze(Iterable) is called.

    Brad Kimmel
