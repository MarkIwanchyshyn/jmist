        public PiecewiseLinearProbabilityDensityFunction(double[] x,

        Creates a new PiecewiseLinearProbabilityDensityFunction.
        Parameters:
        x - The domain points (must be sorted in ascending order). Values less than x[0] or greater than x[x.length - 1] will have a probability density of zero.
        pdf - The values of the probability density function corresponding to the domain points x[i] (must be the same length as x and all values must be non-negative). These values will be normalized to integrate to one.

        java.lang.IllegalArgumentException - if x.length != pdf.length.
        java.lang.IllegalArgumentException - if x is not sorted in ascending order.
        java.lang.IllegalArgumentException - if pdf has negative elements.

        Description copied from interface: ProbabilityDensityFunction
        Generates the domain value at which the cumulative density function evaluates to the specified value.
        Parameters:
        seed - The value of the cumulative density function. This is expected to fall in [0, 1). The results are undefined otherwise.
