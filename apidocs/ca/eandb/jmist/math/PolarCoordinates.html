      • isCanonical

        public boolean isCanonical()
        Determines if this vector is in canonical form. Two equivalent PolarCoordinates (i.e., pointing in the same direction and having the same radius) will have equal values for the angle and the radius when converted to canonical form. For example, the following polar coordinates:

        (Math.PI / 6, 1.0)
        (19 * Math.PI / 6, -1.0)

        represent the same vector. The first of those vectors is in canonical form. The second is not. A PolarCoordinates vector is considered to be in canonical form if -PI <= angle < PI, and radius >= 0.
        Returns:
        A value indicating if this vector is in canonical form.

      • canonical

        public PolarCoordinates canonical()
        Computes the canonical form of this vector. Two equivalent PolarCoordinates (i.e., pointing in the same direction and having the same radius) will have equal values for the angle and the radius when converted to canonical form. For example, the following polar coordinates:

        (Math.PI / 6, 1.0)
        (19 * Math.PI / 6, -1.0)

        represent the same vector. The first of those vectors is in canonical form. The second is not. A PolarCoordinates vector is considered to be in canonical form if -PI <= angle < PI, and radius >= 0.
        Returns:
        The canonical SphericalCoordinates representation of this vector.
