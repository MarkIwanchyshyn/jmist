Class LinearMatrix3

  • java.lang.Object
    • ca.eandb.jmist.math.LinearMatrix3

    public final class LinearMatrix3
extends java.lang.Object
implements java.io.Serializable
    A 4x4 matrix of the form
    abc0
    def0
    ghi0
    0001
    for applying three dimensional linear transformations. This class is immutable.
