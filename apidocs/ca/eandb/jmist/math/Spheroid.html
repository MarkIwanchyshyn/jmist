      • characterize

        public double characterize(Point3 p)
        Evaluates the characteristic function of this Spheroid at the specified point. The characteristic function, f, satisfies:
        • f(p) = 0 if p is on the surface of the spheroid,
        • f(p) < 0 if p is inside the spheroid, and
        • f(p) < 0 if p is outside the spheroid.
        Parameters:
        p - The Point3 at which to evaluate the characteristic function.
        Returns:
        The value of the characteristic function at p.

      • getMatrixRepresentation

        public Matrix4 getMatrixRepresentation()
        Gets the Matrix4 representation of this Spheroid. The Matrix4 returned, A, will satisfy:
        • xtAx < 0 whenever x = (x y z 1)t lies inside this Spheroid,
        • xtAx > 0 whenever x lies outside this Spheroid, and
        • xtAx = 0 whenever x lies on the surface of this Spheroid.
        Returns:
        The Matrix4 representation of this Spheroid.
