      • create

        public static HPoint2 create(double x,
             double y,
             double w)
        Creates a homogenized point.
        Parameters:
        x - The value of the x-coordinate.
        y - The value of the y-coordinate.
        w - The value of the w-coordinate (must be one or zero).
        Returns:
        An HPoint2 corresponding to the given coordinates. If w == 0, a Vector2 is returned. If w == 1, a Point2 is returned.
        Throws:
        java.lang.IllegalArgumentException - if w is not within MathUtil.EPSILON of 0.0 or 1.0.
        See Also:
        MathUtil.EPSILON, Point2, Vector2
