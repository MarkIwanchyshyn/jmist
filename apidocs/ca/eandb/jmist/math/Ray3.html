        public Ray3(Point3 p,
    HPoint3 q)
        Creates a Ray3 from one point along a direction or to another point.
        Parameters:
        p - The Point3 at the origin of the ray.
        q - The HPoint3 indicating the direction and limit of the ray. If q is a Point3, this creates a finite ray from p to q. If q is a Vector3, this creates an infinite ray from p in the direction indicated by q.

      • create

        public static Ray3 create(HPoint3 p,
          HPoint3 q)
        Creates a Ray3 spanning two HPoint3s. If both are Point3s, this is equivalent to new Ray3((Point3) p, (Point3) q). If one is a Point3 and one is a Vector3, this is equivalent to new Ray3((Point3) thePoint, (Vector3) theVector). If both are Vector3s, the return value is null.
        Parameters:
        p - The first HPoint3.
        q - The second HPoint3.
        Returns:
        A Ray3 spanning p and q, or null if p and q are both Vector3s.
        See Also:
        Point3, Vector3, Ray3(Point3, Point3), Ray3(Point3, Vector3)
