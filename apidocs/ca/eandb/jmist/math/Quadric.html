Class Quadric

  • java.lang.Object
    • ca.eandb.jmist.math.Quadric
  • All Implemented Interfaces:
    java.io.Serializable

    public final class Quadric
extends java.lang.Object
implements java.io.Serializable
    Represents a quadric surface, i.e., the solution to xtQx = 0, where x = (x y z 1)t and Q is the 4x4 symmetric matrix characterizing the surface.
    Author:
    Brad Kimmel
    See Also:
    Quadric (wikipedia), Quadratic Surface (MathWorld), Serialized Form

      • characterize

        public double characterize(Point3 p)
        Evaluates the characteristic function f(p) = ptQp, where p = (x y z 1)t. The characteristic function has the following properties:
        • f(p) = 0 if p is on the surface,
        • f(p) < 0 if p is inside the surface, and
        • f(p) > 0 if p is outside the surface.
        Equivalent to this.characteristic(p.toVector4()).
        Parameters:
        p - The Point3 at which to evaluate the characteristic function.
        Returns:
        A value characterizing the p with respect to this Quadric.
        See Also:
        characterize(Vector4), HPoint3.toVector4()

      • gradient

        public Vector4 gradient(Vector4 v)
        Evaluates the gradient of the characteristic function. If v lies on the surface of this Quadric, then the gradient is normal to the surface.
        Parameters:
        v - The Vector4 at which the gradient is to be evaluated.
        Returns:
        The Vector4 of the derivatives with respect to each component.
        See Also:
        characterize(Vector4)

      • gradient

        public Vector3 gradient(Point3 p)
        Evaluates the gradient of the characteristic function. If p lies on the surface of this Quadric, then the gradient is normal to the surface.
        Parameters:
        p - The Point3, p = (x y z 1)t, at which the gradient is to be evaluated.
        Returns:
        The Vector3 of the derivatives with respect to each component. The fourth component is ignored.
        See Also:

      • normalAt

        public Vector3 normalAt(Point3 p)
        Determines the normal to this Quadric surface at the specified point.

        Equivalent to this.gradient(p).unit().
        Parameters:
        p - The Point3 on the surface at which to evaluate the normal.
        Returns:
