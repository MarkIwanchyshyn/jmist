      • toString

        public java.lang.String toString(java.lang.String variableName)
        Generates a string representation of this Polynomial, with the specified variable name as the unknown quantity. It is the responsibility of the caller to place parentheses around the variable if necessary (for example this.toString("xy") would generate something like "1.0xy^0 - 2.0xy^1 + 3.0xy^2", when "1.0(xy)^0 - 2.0(xy)^1 + 3.0(xy)^2" is what was intended; this.toString("(xy)") would achieve this).
        Parameters:
        variableName - The name of the unknown quantity.
        Returns:
        The String representation of this Polynomial with variableName for the unknown quantity.
