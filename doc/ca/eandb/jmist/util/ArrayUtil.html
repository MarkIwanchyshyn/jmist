        public static double[] fillRange(double[] array,
                 double first,
                 double last)
        Fills an array with values uniformly within a specified range.
        Parameters:
        array - The array to populate.
        first - The value to assign to the first element of array.
        last - The value to assign to the last element of the array.
        Returns:
        A reference to array, populated with the values array[i] == first + (i / (array.length - 1)) * last, for 0 <= i < array.length.

      • fillRange

        public static int[] fillRange(int[] array,
              int first,
              int last)
        Fills an array with each value within a specified range.
        Parameters:
        array - The array to populate.
        first - The value to assign to the first element of array (must be null or of length equal to 1 + |last - first|.
        last - The value to assign to the last element of the array.
        Returns:
        A reference to array, populated with the values first through last.
        Throws:
        java.lang.IllegalArgumentException - if array != null and array.length != 1 + Math.abs(last - first).
