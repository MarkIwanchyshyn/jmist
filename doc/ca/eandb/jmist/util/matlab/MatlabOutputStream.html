        public void beginArrayElement(java.lang.String name,
                     MatlabArrayType arrayType,
                     MatlabDataType dataType,
                     boolean complex,
                     boolean global,
                     boolean logical,
                     int[] dimensions,
                     int elements)
                       throws java.io.IOException
        Begins a numeric or character array element.
        Parameters:
        name - The name of the MATLAB array.
        arrayType - The MatlabArrayType of the array.
        dataType - The MatlabDataType to use to store the elements of the MATLAB array.
        complex - A value indicating whether the imaginary part is to be included.
        global - A value indicating whether the array is to have global scope.
        logical - A value indicating whether the array is to be used for logical indexing.
        dimensions - An array describing the dimensions of the array (there must be at least two dimensions). This may be null, in which case the array will have the dimensions { elements, 1 }.
        elements - The total number of elements in the array (must be equal to the product of the dimensions).
        Throws:
        java.io.IOException - if writing to the underlying stream fails.
        java.lang.IllegalArgumentException - if elements is not equal to the product of the dimensions.
        java.lang.IllegalArgumentException - if dimensions.length < 2.
