Class LinearMatrix2

  • java.lang.Object
    • ca.eandb.jmist.math.LinearMatrix2
  • All Implemented Interfaces:
    java.io.Serializable

    public final class LinearMatrix2
extends java.lang.Object
implements java.io.Serializable
    A 3x3 matrix of the form
    ab0
    cd0
    001
    for applying two dimensional linear transformations. This class is immutable.
    Author:
    Brad Kimmel
