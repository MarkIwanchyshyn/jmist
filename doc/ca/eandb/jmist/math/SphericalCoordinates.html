      • isCanonical

        public boolean isCanonical()
        Determines if this vector is in canonical form. Two equivalent SphericalCoordinates (i.e., pointing in the same direction and having the same radius) will have equal values for both angles and the radius when converted to canonical form. For example, the following spherical coordinates:

        (Math.PI / 6, Math.PI / 2, 1.0)
        (-5 * Math.PI / 6, Math.PI / 2, -1.0)

        represent the same vector. The first of those vectors is in canonical form. The second is not. A SphericalCoordinates vector is considered to be in canonical form if 0 <= polar <= PI, -PI <= azimuthal < PI, and radius >= 0.
        Returns:
        A value indicating if this vector is in canonical form.
