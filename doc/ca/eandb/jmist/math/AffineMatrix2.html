Class AffineMatrix2

  • java.lang.Object
    • ca.eandb.jmist.math.AffineMatrix2
  • All Implemented Interfaces:
    java.io.Serializable

    public final class AffineMatrix2
extends java.lang.Object
implements java.io.Serializable
    A 3x3 matrix of the form
    abx
    cdy
    001
    for applying two dimensional affine transformations. This class is immutable.

    Brad Kimmel
