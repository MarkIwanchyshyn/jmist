      • create

        public static HPoint3 create(double x,
             double y,
             double z,
             double w)
        Creates a homogenized point.
        Parameters:
        x - The value of the x-coordinate.
        y - The value of the y-coordinate.
        z - The value of the z-coordinate.
        w - The value of the w-coordinate (must be one or zero).
        Returns:
        An HPoint3 corresponding to the given coordinates. If w == 0, a Vector3 is returned. If w == 1, a Point3 is returned.
        Throws:
        java.lang.IllegalArgumentException - if w is not within MathUtil.EPSILON of 0.0 or 1.0.
        See Also:
        MathUtil.EPSILON, Point3, Vector3

      • create

        public static HPoint3 create(Vector4 v)
        Creates a homogenized point.
        Parameters:
        v - The Vector4 indicating the coordinate values.
        Returns:
        An HPoint3 corresponding to the given coordinates. If v.w() == 0, a Vector3 is returned. If v.w() == 1, a Point3 is returned.
        Throws:
        java.lang.IllegalArgumentException - if v.w() is not within MathUtil.EPSILON of 0.0 or 1.0.
        See Also:
        MathUtil.EPSILON, Point3, Vector3, Vector4, Vector4.w()
