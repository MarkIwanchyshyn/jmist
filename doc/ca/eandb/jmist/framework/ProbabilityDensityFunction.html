        Parameters:

      • warp

        double[] warp(double[] seeds,
            double[] results)
        Generates the domain values at which the cumulative density function evaluates to the specified values.
        Parameters:
        seeds - The values of the cumulative density function (must not be null). The values are expected to fall in [0, 1). The results are undefined for seed values outside this range.
        results - An optional preallocated array to populate with the results (if non-null, must be the same length as seeds).
        Returns:
        C-1(seeds[i]) for 0 <= i < seeds.length, where C(x) is the integral from zero to x of P(t)dt, and P(t) is the value of this ProbabilityDensityFunction at t.
        Throws:
        java.lang.IllegalArgumentException - if results != null and results.length != seeds.length.
        See Also:
        evaluate(double), warp(double)

      • evaluate

        Evaluates this probability density function at the specified points in the domain.
        Parameters:

        Returns:
