    public abstract class IntersectionDecorator
extends java.lang.Object
implements Intersection
    An abstract Intersection that decorates another (e.g., by applying a transformation). Default implementations are provided that call the corresponding methods on the decorated Intersection. It is up to the derived class to override whichever methods are necessary to provide the specific decoration required.
