        public RasterJob(ColorModel colorModel,
         PixelShader pixelShader,
         Display display,
         int width,
         int height,
         int cols,
         int rows)
        Creates a new RasterJob. This job will divide the image into rows * cols tasks to render roughly equally sized chunks of the image.
        Parameters:
        colorModel - The ColorModel to use to render the image.
        pixelShader - The PixelShader to use to compute the values of individual Pixels.
        display - The Display to record the image to.
        width - The width of the rendered image, in pixels.
        height - The height of the rendered image, in pixels.
        cols - The number of columns to divide the image into.
        rows - The number of rows to divide the image into.
