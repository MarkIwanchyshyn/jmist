      • add

        public static Color add(Color a,
        Color b)
        Adds two Colors, one or both of which may be null.
        Parameters:
        a - The first Color.
        b - The second Color.
        Returns:
        If a != null and b != null, then the sum a.plus(b) is returned. If one is non-null and the other is null, the non-null value is returned. If both are null, null is returned.
        See Also:
        Color.plus(Color)

      • sub

        public static Color sub(Color a,
        Color b)
        Subtracts two Colors, one or both of which may be null.
        Parameters:
        a - The Color to subtract from.
        b - The Color to subtract.
        Returns:
        If a != null and b != null, then the difference a.minus(b) is returned. If b == null, then a is returned. If a == null, then b.negative() is returned. If both are null, then null is returned.
        See Also:
        Color.minus(Color), Color.negative()
