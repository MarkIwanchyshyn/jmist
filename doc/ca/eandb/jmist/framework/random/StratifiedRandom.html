ca.eandb.jmist.framework.random

Class StratifiedRandom

  • java.lang.Object
    • ca.eandb.jmist.framework.random.StratifiedRandom
  • All Implemented Interfaces:
    Random, java.io.Serializable

    public final class StratifiedRandom
extends java.lang.Object
implements Random
    A random number generator that stratifies the results into a number, n, of equally sized divisions of [0, 1). The random number generator is guaranteed to generate exactly one value in each interval [i/n, (i+1)/n) for each block of n successive calls to next().
