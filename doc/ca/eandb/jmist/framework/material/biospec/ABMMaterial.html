Class ABMMaterial

    public final class ABMMaterial
extends OpaqueMaterial
    A Material implementing the ABM-U/ABM-B light transport models.

    The algorithm implemented here is described in:

    • G.V.G. Baranoski, Modeling the interaction of infrared raditaion (750 to 2500 nm) with bifacial and unifacial plant leaves, Remote Sensing of Environment 100:335-347, 2006.
    • G.V.G. Baranoski, D. Eng, An investigation on sieve and detour effects affecting the interaction of collimated and diffuse infrared radiation (750 to 2500 nm) with plant leaves, IEEE Transactions on Geoscience and Remote Sensing 45(8):2593-2599, August 2007.
    Author:
    Brad Kimmel
    See Also:
    Run ABM-U Online, Run ABM-B Online, Serialized Form

                   Vector3 v,
                   boolean adjoint,
                   WavelengthPacket lambda,
                   double ru,
                   double rv,
                   double rj)
        Description copied from interface: Material
        Scatters a ray incident on this material.
        Specified by:
        scatter in interface Material
        Overrides:
        scatter in class AbstractMaterial
        Parameters:
        x - The SurfacePoint that the ray is incident upon.
        v - The Vector3 indicating the direction of travel of the incident ray (note that the Vector3 points toward the surface).
        adjoint - A value indicating whether the path being simulated is in the reverse direction from the direction in which light is traveling (i.e., true for path tracing from the eye, false for light tracing).
        lambda - The WavelengthPacket associated with the light traveling along this path.
        ru - The first random variable (must be in [0, 1]).
        rv - The second random variable (must be in [0, 1]).
        rj - The third random variable (must be in [0, 1]).
        Returns:
        The ScatteredRay, or null to indicate that the ray was absorbed.
