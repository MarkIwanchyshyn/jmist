    public final class VarianceTargetPixelShader
extends java.lang.Object
implements PixelShader
    A pixel shader decorator that averages the results of another pixel shader. The number of samples used may vary within a specified range based on a provided target variance.

        Creates a new VarianceTargetPixelShader.
        Parameters:
        varianceTarget - The target variance. Sampling will stop once the variance drops below this level.
        minSamples - The minimum number of samples to use.
        maxSamples - The maximum number of samples to use.
        checkInterval - The number of samples to compute between variance tests.
        testMode - A value indicating whether to render in test mode. In test mode, the color of the pixel returned is a greyscale value indicating how many samples were used (black represents the minimum number of samples were used, white indicates that the maximum number of samples were used).
        pixelShader - The PixelShader to use to gather samples.
