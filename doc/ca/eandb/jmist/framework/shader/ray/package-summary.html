  • Class Summary 
    Class Description
    AngularEnvironmentRayShader
    A RayShader that shades a ray based on a Texture2 based on an angular projection.
    DistanceRayShader
    A ray shader that shades a ray according to the distance to the nearest intersection along the ray.
    HemisphericalEnvironmentRayShader
    A RayShader that shades a ray based on a Texture2 representing a hemispherical environment map.
    SceneRayShader
    A RayShader that shades a ray using a scene description.
    SphericalEnvironmentRayShader
    A RayShader that shades a ray based on a Texture2 representing a spherical environment map.
    UniformRayShader
    A RayShader that always samples the same Spectrum.
    VisibilityRayShader
    A ray shader that shades based on the evaluation of a visibility function.
